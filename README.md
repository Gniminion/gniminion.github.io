# Personal Portfolio Website

A modern, responsive portfolio website built with React and Vite.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Stack

- React
- Vite
- Tailwind CSS
